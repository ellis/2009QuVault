Some questions in QuVault require you to write your answers in Matlab-style. Here are some important examples:

Operators

x = 100Set the value of the variable x
y = x + 10Addition
y = x - 10Subtraction
y = x * 2Multiplication
y = x / 2Division
y = x^2Powers
y = sqrt(x)Square root
y = 1/(1/4 + x^2.5) - 1A more complicated expression using parentheses

Multiple equations

Sometimes you'll need to enter in more than one equation. Equations can be separated either by placing them on separate lines, or you can separate equations with a semicolon.

x = 100
y = sqrt(x)		Multiple assignments separated by ENTER
x = 1; y = 2Multiple assignments separated by a semicolon

Checking your syntax

You can check whether you wrote valid matlab-style code by clicking the [Validate] button at the bottom of the work pages.
